# docs/research

Date-tagged research and decision documents that provide context to humans and AI agents. Often tied to [ADRs](../adr/README.md).

## Rules

| Rule | Reason |
| :--- | :----- |
| Filenames start with an ISO date | Chronological sorting; staleness detection |
| Documents are immutable | Prevents intra-document inconsistencies. New info → new document |
| Documents can be deleted | Stale research may be misleading |
